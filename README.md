# 課題
- Todoリストアプリ
## 概要
- Todoリストを作成、管理するWindowsアプリ。
  - (WindowsForm / WPF のいづれかで作成してください。)
- Todoの登録、完了、削除、一覧表示を行うことができる。

## 作業環境
- Visual Studio / Visual Studio Code
- C#
- WindowsForm / WPF
- .Net Framework / .NET

以下は、機能仕様

### 機能
- Todoの登録
- Todoの完了
- Todoの削除
- Todoの一覧表示
- Todoの一覧表示(完了済み)
- Todoの一覧表示(未完了)

### 画面
- Todoの一覧表示
- Todoの一覧表示(完了済み)
- Todoの一覧表示(未完了)
- Todoの登録

### Todoデータ
- id(自動採番)★
- タイトル(必須)★
- 内容(任意) (1000文字まで)

### データ保存
- 任意の形式、ただし、インストールが必要な環境は不可。
- SQLServerのインストールが必要な環境は不可。
- レジストリへの書き込みは不可。

### 完了条件
- アプリを起動できる。
- Todoを登録できる。
- Todoを完了できる。
- Todoを削除できる。
- Todoを一覧表示できる。
- Todoを一覧表示(完了済み)できる。
- Todoを一覧表示(未完了)できる。
- Todoのデータは、テキストファイル / データベースに保存される。
- Todoのデータは、アプリを再起動しても復元される。
